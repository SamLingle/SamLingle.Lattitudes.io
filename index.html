Project Summary

This project seeks to answer the question: What Is the weather like as we approach the equator? To answer this question we used a series of python requests through the API 'https://openweathermap.org/(turn this into a link), and JSON traversal commands. These requests gathered weather data collected from over 500 randomly selected cities of varying distances from the equator. The data was then codified and visualized as a series of charts which are presented throughout the pages of this website. In the Navigation Bar you will find links to the individual charts, the charts side by side for comparison, and the data used to construct these charts. Thank you for visiting this site!

Specifics: I used the CityPy library to select cities closest to the randomly generated lattitue and longitudinal pairings. over 600 cities were selected, but only the cities that were able to produce a record of the weather were used, and any duplicates were dropped. The usable data was pulled into a CSV file and converted into a DataFrame using the Pandas library. That dataframe was then run through with PyPlots to

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
